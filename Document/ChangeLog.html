Version 0.05 - 2012/12/05

Library

  • 機能追加・変更
    • 【XG3D】機能が制限されたモデルの描画機能を追加。
    • 【XBase】モジュール『数学』に Vector3i Vector4i を追加。
    • 【XBase】モジュール『ユーティリティ』に Hash を追加。
    • 【XBase】Matrix34 に toQuaternion() を追加。
    • 【XBase】Quaternion に toShortString() を追加。
    • 【XBase】Vector2 Vector2i Vector3 Vector4 のコンストラクタバリエーションを追加。
  • バグ修正
    • 【XBase】Vector2i::toShortString() が正しい値を返さない不具合を修正。
    • 【XBase】AABBox2i::toAABB2i() のterm値が意図通りではなかった不具合を修正。
    • 【XBase】Color4::toRGBAb() が正しい値を返さない不具合を修正。
    • 【XBase】Color4b::toRGBAf() が正しい値を返さない不具合を修正。
    • 【XBase】ResFileStream::open() で常にアスキーモードでファイルを読み込んでいた不具合を修正。
    • 【XBase】Matrix34::LookAt() Matrix44::LookAt() が正しい値を返さない不具合を修正。

CommandLineTool

  • 機能追加・変更
    • 【DAEConverter】新規追加。
    • 【XG3DConverter】出力するバイナリファイルの仕様を変更。（※Version 0.04以前の互換性はありません）
    • 【XG3DConverter】コマンドライン引数の仕様を変更。
  • バグ修正
    • 【XDataConverter】usage表記中のutf_8をutf-8に修正。

CLRDLL

  • 機能追加・変更
    • 【CrossFramework.XG3D】XMLの仕様を変更。（※Version 0.04以前の互換性はありません）

Version 0.03 - 2011/04/29

Library

  • 機能追加・変更
    • 【XBase】モジュール『型』に Bool Enum を追加。
    • 【XBase】モジュール『ユーティリティ』に FrameCounter Color4b を追加。
    • 【XBase】モジュール『数学』に AABBox2 AABBox2i Vector2i を追加。
    • 【XBase】モジュール『メモリ』に AutoPtr AutoSPPtr ScopedPtr を追加。
    • 【XBase】BitSet をクラス型に変更。従来の BitSet は BitSetPOD に変更。
    • 【XBase】BitSet アライメント値指定対応。
    • 【XBase】MacOSX で .nib ファイルがなくても起動できるように対応。
    • 【XBase】Windows MacOSX DisplayのサイズをiOSと同じ480x320に変更。
    • 【XG3D】ResTex の対応フォーマットを追加。
    • 【XG3D】TexSetting に設定項目を追加。
    • 【XG3D】実行時にテクスチャのイメージを作成・編集する RuntimeTex を追加。
    • 【XG3D】デモ『runtimetex』を追加。

CommandLineTool

  • 機能追加・変更
    • 【TGAConverter】フォーマット指定(rgb8 rgb565 rgba8 rgb5a1 rgba4444 a8 la8 l8)対応。

CLRDLL

  • 機能追加・変更
    • 【CrossFramework.XG3D】XTEX.xsd フォーマットの項目の要素を追加。
  • バグ修正
    • 【全体】CLRDLLフォルダが消えていた不具合を修正。
