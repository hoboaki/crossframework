CrossFramework

CrossFrameworkはWindows，MacOSX，iOSをターゲットとしたゲームアプリケーション開発のツール・ライブラリです。

内容

ReadMe.html - このファイル。
ChangeLog.html - 変更履歴。（HTML）
CommandLineTool - コマンドラインツール。
CLRDLL - CLRアプリケーション用のDLL。
Library - ランタイム用のC++ライブラリ。（リファレンスマニュアルHTML）

動作環境

  • ランタイム環境
    • Windows7
    • MacOSX(10.7以上)
    • iOS(6.1以上)
  • ビルド環境
    • Windows7 - Microsoft VisualC++ 2010 ExpressEdition
    • MacOSX 10.8.5 - XCode4.6.3
    • ※以上は作者のビルド環境でありこれ以外の環境でも動く可能性はあります
  • CLR環境
    • Windows - Microsoft .NET Framework 4
    • MacOSX - Mono2.10.11
    • ※以上は作者のCLR環境でありこれ以外の環境でも動く可能性はあります
