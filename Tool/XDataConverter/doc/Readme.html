1. 概要

XDataはXMLに記述されたデータをバイナリデータに変換し、C++言語で記述されたプログラムからアクセスするツールおよびライブラリ群です。

2. 内容

Readme.html - このドキュメントファイル。
cpplib/include/... - C++ライブラリファイル群。
schema/XData.xsd - XMLファイルのスキーマファイル。
xdata_conv/xdata_conv.exe - バイナリコンバータ。

3. XMLファイルのフォーマット

データの内容はXMLファイルに記述します。
次のようなフォーマットで記述してください。

<?xml version="1.0" encoding="utf-8"?>
<xdata_root
xmlns="http://10106.net/xdata"
xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
xsi:schemaLocation="
http://10106.net/xdata ../../schema/XData.xsd
"
major_version="1"
minor_version="0"
>
<reference label_name="pos_array"/>
<label name="pos_array"/>
<sint8_array count="3">-128 127 1</sint8_array>
<uint8 value="0"/>
<uint16 value="0"/>
<uint32 value="0"/>
<uint64 value="0"/>
<sint8 value="0"/>
<sint16 value="0"/>
<sint32 value="0"/>
<sint64 value="0"/>
<float32_array count="4">1.0 0.5 0.25 <!-- comment --> 0 </float32_array>
<float32 value="3.1415926534"/>
<float64_array count="0"></float64_array>
<float64 value="10000.2"/>
<string value="test"/>
<align value="32"/>
</xdata_root>

詳細はスキーマ（schema/XData.xsd）を参照してください。（説明はしょってごめんなさい･･･）

4. バイナリコンバータ

xdata_conv/xdata_conv.exeは、XMLファイルをバイナリファイルに変換するコマンドラインアプリケーションです。
.Netframework2.0を使用しています。Windows上で実行する場合は、.Netframework2.0以上のランタイムをインストールしてください。
他のOSでは、monoなどのランタイムをインストールして使用してください。

引数のオプションなどは、アプリケーションを実行して確認してください。（説明はしょってすみません･･･）
